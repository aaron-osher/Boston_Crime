# EDA Final Project
## Boston Crime Analysis

Exploratory Data Analysis final project

Boston Crime Dataset - 
(https://data.boston.gov/dataset/crime-incident-reports-august-2015-to-date-source-new-system)

The data contains Boston crime data. Location as well as the specific crime the occured between 2015 and 2019. It is provided by the Boston government. 

We are trying to analyze the crimes that happened in Boston by district type so we can suggest people avoid certain areas based on the crimes that happened there. You can use this visualization to see what streets are safer to walk at night.
Local people and non-locals don't know where specific crimes happen just maybe they hear about it in the news. We can use this visualization to see the actual location of crimes that happened  recently instead of only having a limited understanding that crimes happen at all.

Boston District list - 
(https://www.boston.gov/departments/police)
